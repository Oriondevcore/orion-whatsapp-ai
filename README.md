# Orion WhatsApp AI

Naledi-powered WhatsApp chatbot for South African businesses. Built with Cloudflare Workers and Hono.

## What It Does

- Responds to customer enquiries automatically via WhatsApp
- Books appointments, answers FAQs, routes to humans when needed
- Learns from each conversation
- Works 24/7 without human supervision

## Architecture

WhatsApp Gateway (Puppeteer) ←→ Workers API ←→ AI Router (Z.AI / Cloudflare) ←→ D1 Database
